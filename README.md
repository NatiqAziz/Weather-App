# ⛅ Weather App

**Get real-time weather for any city worldwide.**

## Live Demo
https://openweather-web-app.netlify.app/

---

## Features

🌍 Search any city globally  
🌡️ Current temperature and conditions  
💧 Humidity levels  
🎨 Clean, responsive UI  

## Tech

HTML • CSS • JavaScript • OpenWeatherMap API

## Setup

Get API key from: https://openweathermap.org/api  
Replace `YOUR_API_KEY_HERE` in `app.js` with your key.
